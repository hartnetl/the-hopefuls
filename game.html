How to play
Well, well, well. You're the type that got guts, aren't ya? If you're sure you can handle the spooks, here's how to play this game! Click on which story you want, read the paragraph (watch out for the creepies), make a choice as to where you want to go from the options below the story. If you are too scared to follow it through click the game name in the top left corner to return safely home.
